# 🗺️ Roadmap — NEXUS

## Phase 1 — MVP (Mois 1–6)

**Objectif** : version fonctionnelle utilisable en beta privée par 10 000 personnes.

- [x] Architecture initiale
- [x] Schéma DB
- [x] Auth + identités fragmentées (facets)
- [x] Cercles concentriques
- [x] Posts texte + médias
- [x] Time-Capsules
- [x] Wallet (économie inversée)
- [x] Marketplace P2P basique
- [x] Crisis detection v1 (lexique)
- [x] Carbon-cost calculator
- [ ] Notifications push (Firebase, APNS)
- [ ] Modération IA basique (text moderation API)
- [ ] App mobile React Native (iOS + Android)
- [ ] Beta privée 10 000 users (FR)

## Phase 2 — Lancement Public EU (Mois 7–12)

- [ ] Messagerie E2E (Signal protocol)
- [ ] Vidéo (upload, transcoding, streaming)
- [ ] Audio spatial
- [ ] Marketplace : escrow, dispute resolution, payment integration (Stripe)
- [ ] Crisis service : équipe humaine + partenariats associations
- [ ] Compliance RGPD complète + audit indépendant
- [ ] Multi-langues : EN, ES, DE, IT, PT, PL
- [ ] Lancement public France → UE

## Phase 3 — Tokenisation & Web3 (Mois 13–18)

- [ ] Wallet on-chain (couche L2 type Polygon/Base)
- [ ] Tokens créateurs (ERC-20 + smart contracts)
- [ ] NFT pour contenus rares (collections, drops)
- [ ] DAO de modération (vote pondéré par réputation)
- [ ] Mode AR (filtres, sticker placements 3D)
- [ ] Expansion : Amérique du Sud, Asie du Sud-Est

## Phase 4 — Expansion Globale (Mois 19–24)

- [ ] 50+ langues supportées
- [ ] Fédération inter-instances (ActivityPub compatible)
- [ ] Mode "Vraie Vie" avancé (gamification éducative)
- [ ] Lancement campagne anti-addiction & santé mentale
- [ ] Outils Business (analytics, scheduling, API publique)
- [ ] Partenariats institutionnels (universités, ONG, gouvernements)

## Phase 5 — Maturité (An 3+)

- [ ] 1 milliard d'utilisateurs
- [ ] IPO ou structure coopérative (à débattre démocratiquement)
- [ ] Infrastructure verte 100% (data centers solaires/géo)
- [ ] Programme de financement de la recherche en santé mentale numérique
- [ ] Standard ouvert pour les "social tokens" interopérables
- [ ] 4 milliards d'utilisateurs (an 5–7)
